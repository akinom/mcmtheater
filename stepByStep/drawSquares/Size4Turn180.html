The Actor now alternates between drawing a square with 50, 60, 70, and 80 pixels side length. It still turns 180 degrees between drawing squares.

The following code snippets show several other ways to achieve the same behaviour. The last snippet, which uses a for loop, is the code furher improvements are based on.

In the last change we formulate a for loop. The loop executes the statements inside its curly braces ({}) several times. len receives the value 20 before the statements inside are executed for the first time, see for(double len = 20; .... After each execution of the statements inside the loop len is incremented by 10, see for(...; len = len + 10). The loop finishes execution once the variable len is no more smaller or equal to 50.

Thus the statements inside the loop are executed with len containing a value of 20 in the first 'round', 30 in the second, 40 in the third, and 50 in the fourth. After the fourth execution of the loop's statements len receives a value of 60 and since 60 is no more smaller or equal to 50 the loops finishes.
