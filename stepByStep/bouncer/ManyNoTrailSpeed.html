This final example is a slight variant of the previous one. It sets the debug mode of all actors to true by calling the setDebugMode method. An actor in debug mode changes its direction incrementally, which gives the balloons a more natural look when they bounce off the stage's borders.

This example actor differs from the previous in that it assigns different step sizes to the Bouncers that it creates. Bouncers like all Actors in mcmTheater move about the Stage incrementally, step by step. The size of these steps defines each Actor's speed: the bigger the step size the faster the Actor, see

The initial actor here uses the convenience method Dice.throwDouble(0.5, 5.0) which computes a random value bewteen 0.5 and 5.0. Thus b.setStepSize(Dice.throwDouble(0.5, 5.0)); sets the step size randomly to a value between 0.5 and 5.0.
