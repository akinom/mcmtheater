Since the initial actor uses a for loop to create pairs of green and yellow bouncing ballons it is very easy to change it to create any number of pairs. The actor below chooses to create ten pairs by changing the number 4 used in the previous example's loop to the number 10 in its own the loop. The loop again starts its first execution with the variable i set to 1 and goes on by increasing that value by 1 for each repetition. Thus i becomes 2, 3, 4, ..., 10. The loop finishes when i becomes bigger than 10.
