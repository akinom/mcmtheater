The shows various actors that jump all over the stage performing a simple task at random locations. This and the next page's initial actors show how to start multiple copies of these actors to execute simultaneoulsy on the stage. They create actors with different properties such as varying trail or shape colors or different kinds of shapes. The initial actors below terminate afterwards leaving the stage to the jumping actors.

The links below the applets lead to the complete Java source code of the initial actors as well as the source code of the actors that they create and start.

This initial actor executes its for loop twenty times creating a Jump actor each time. The created actors receive a circle shape with a random color. Each new Jump actor's delay variable is set to a randomly chosen number between 5 and 25. Choosing different delays forces the Jump actors to jump at different intervals since they use their delays to determine how many ticks on the stage's clock they need to sleep between jumps.
