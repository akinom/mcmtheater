The first actor below jumps 10 pixels forward before drawing a triangle and keeps doing so forever even when it reaches the edge of the stage.

The second actor is a little smarter. It jumps and draws triangles only as long as it stays within the stage's boundaries, see while (getStage().isInside(getPos())) { ... }. It draws rotated triangles by turning 30 degrees to the left before calling its triangle drawing shape method. It turns 30 degrees right afterwards so that its directions is again the same as when it started acting. Thus it always jumps foward in the same direction drawing its triangles along a straight line.

The following actor is a variant of the one above. It jumps to a random position on the stage followed by a call to the shapeLine method, which draws a series of triangles until reaching the stage's edge. The shapeLine() method contains the same while (getStage().isInside(getPos())) { ... } loop as the previous actors act() method.

The shows how multiple copies of this last actor can be started so that they execute in parallel on the stage.
